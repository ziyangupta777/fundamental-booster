# Fundamental Booster

A beginner-friendly Python program that demonstrates fundamental programming concepts including user input, data types, type conversion, and basic calculations.

## Overview

This program showcases essential Python fundamentals by collecting user information and performing basic operations and type conversions on that data.

## Features

- **User Input**: Accepts name (string), age (integer), and height (float) from user
- **Data Type Demonstrations**: Shows how to work with different data types (str, int, float)
- **Type Conversion**: Demonstrates converting between different data types
- **Calculations**: Calculates birth year based on current year (2026) and age
- **Object Inspection**: Shows type checking and memory address identification

## How to Run

1. Ensure Python is installed on your system
2. Open a terminal/command prompt
3. Navigate to the project directory
4. Run the program:
   ```bash
   python "fundamental booster.py"
   ```

## Program Flow

1. Prompts user to enter their name
2. Prompts user to enter their age (converted to integer)
3. Prompts user to enter their height (converted to float)
4. Displays the input information
5. Calculates and displays the birth year
6. Demonstrates type conversions:
   - Converts age to float
   - Converts height to integer
7. Shows the data type of height variable
8. Displays the memory address (id) of height variable

## Concepts Demonstrated

- **Input/Output**: Using `input()` and `print()`
- **Type Conversion**: `int()`, `float()` functions
- **Arithmetic Operations**: Basic calculation (2026 - age)
- **Type Inspection**: Using `type()` function
- **Memory Inspection**: Using `id()` function
- **String Formatting**: Printing formatted output
